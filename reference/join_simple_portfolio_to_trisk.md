# Join simple portfolio to aggregated TRISK outputs

Join simple portfolio to aggregated TRISK outputs

## Usage

``` r
join_simple_portfolio_to_trisk(portfolio_data, trisk_agg)
```

## Arguments

- portfolio_data:

  Aggregated simple portfolio.

- trisk_agg:

  Aggregated TRISK outputs.

## Value

Joined dataframe at company/term/lgd/sector/technology level.
